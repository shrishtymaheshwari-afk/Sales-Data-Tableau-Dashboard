# Sales-Data-Tableau-Dashboard
Interactive Tableau dashboard analyzing Superstore sales, profit, customer segments, and category performance to generate business insights through data visualization.


## Project Overview

This project was developed to transform raw sales data into actionable business insights through interactive visualizations and KPI tracking. The dashboard enables users to explore performance trends across regions, categories, customer segments, and order priorities over a 4-year period.

The main goal of this project was to improve business decision-making through data storytelling and dashboard-driven analytics.

## Business KPIs

* 💰 Total Sales: ₹7.07 Cr
* 📈 Total Profit: ₹49.47 L
* 📦 Total Shipping Cost: ₹7.96 L
* 📉 Average Profit Margin: ~6.8%


##  Key Insights-

### Order Source Performance

Amazon generated the highest overall profit among all order sources with approximately ₹10.06L profit contribution.

### Customer Segment Analysis

* Consumer segment contributed the highest sales volume.
* Corporate segment showed strong profitability, indicating high-value customer potential.

### Category Performance

Office Supplies emerged as the most profitable category with ₹21.46L profit.

### Profit Margin Trends

Profit margins remained consistently stable between 6.75% and 6.90% from 2018 to 2021, reflecting sustainable business growth.
